Incorporar o site Bugu em outro sistema

Use o código abaixo na aplicação que vai exibir o Bugu. Troque URL_DO_BUGU pela URL onde o site Bugu está publicado (ex.: https://meusite.com/bugu/ ou https://bugu.meusite.com).

1. Iframe simples

<iframe
  src="URL_DO_BUGU"
  title="Bugu Delícias"
  width="100%"
  height="800"
  style="border: none;"
></iframe>

2. Iframe com altura dinâmica (recomendado)

O site Bugu envia a altura do conteúdo para o pai. Use um id no iframe e o script abaixo para ajustar a altura automaticamente.

<iframe
  id="iframe-bugu"
  src="URL_DO_BUGU"
  title="Bugu Delícias"
  width="100%"
  height="600"
  style="border: none; display: block;"
></iframe>

<script>
(function() {
  var iframe = document.getElementById('iframe-bugu');
  if (!iframe) return;
  function setHeight(e) {
    if (e.origin !== new URL(iframe.src).origin) return;
    if (typeof e.data === 'object' && e.data.type === 'bugu-height' && e.data.height) {
      iframe.style.height = e.data.height + 'px';
    }
  }
  window.addEventListener('message', setHeight);
})();
</script>

3. Exemplo em React (sistema pai em React)

const BUGU_URL = 'URL_DO_BUGU'; // ex: 'https://meusite.com/bugu/'

function BuguEmbed() {
  const iframeRef = useRef(null);

  useEffect(() => {
    const iframe = iframeRef.current;
    if (!iframe) return;
    const handler = (e) => {
      try {
        if (e.origin !== new URL(BUGU_URL).origin) return;
      } catch (_) { return; }
      if (e.data?.type === 'bugu-height' && typeof e.data.height === 'number') {
        iframe.style.height = e.data.height + 'px';
      }
    };
    window.addEventListener('message', handler);
    return () => window.removeEventListener('message', handler);
  }, []);

  return (
    <iframe
      ref={iframeRef}
      src={BUGU_URL}
      title="Bugu Delícias"
      width="100%"
      height={600}
      style={{ border: 'none', display: 'block' }}
    />
  );
}
